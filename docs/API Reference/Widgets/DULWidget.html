      class DULWidget

      Header:DULWidget.h
      Inherits From:UIView
      Description:

      DULWidget is the base class of all widgets in the UILibrary. It is the default implementation of a UIView that implements the DULWidgetProtocol.

      Class Members:
      property
      aspectRatio
      property aspectRatio
      Header:DULWidget.h
      Description:

      The aspect ratio of the widget in width/height. This value may change. A widget with a variable aspectRatio inside a widget collection will request the collection to invalidate its layout. Default is 1.0.

      method transformValue:forKey
      Header:DULWidgetProtocol.h
      Description:

      This method is called by the UILibrary CoreModel when a value has been received from the drone. It is the entry point in the widget for drone data. @param value A valid DULSDKModelValue object containing the value associated with the DJIKey @param key The original key that was registered as a dependentKey

      Input Parameters:
      nonnull DULSDKModelValue * valuethe value associated with the key.
      nonnull __kindof DJIKey * keythe SDK key as registered in the dependentKey

      interactionExpectationLevel
      property interactionExpectationLevel
      Header:DULWidgetProtocol.h
      Description:

      Defines the level of interaction that is expected from the Widget. For instance, a widget with a None level is not expected to provide any touch handling. On the other hand, a widget with a Full level is expected to provide touch handling and events.

      See Also:

      DULWidgetInteractionExpectionLevel
